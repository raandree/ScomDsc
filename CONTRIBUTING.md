# Contributing to ScomDsc

Thank you for considering contributing to ScomDsc!

Please visit the DSC Community [contributing guidelines](https://dsccommunity.org/guidelines/contributing/)
for full details.

## Building the Module

```powershell
./build.ps1 -ResolveDependency -Tasks build
```

## Running Tests

```powershell
./build.ps1 -Tasks test
```

## HQRM Tests

```powershell
./build.ps1 -Tasks hqrmtest
```
